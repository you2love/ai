什么是人工智能？
AI的核心概念

人工智能（AI），简单来说就是让机器能够像人类一样思考和学习的技术。

🤔 换个说法
┌─────────────────────────────────────────────────────────┐ │ │ │ 传统程序： 输入 → 规则 → 输出 │ │ (固定的逻辑) │ │ │ │ AI程序： 输入 → 学习 → 模型 → 输出 │ │ (从数据中学习规律) │ │ │ └─────────────────────────────────────────────────────────┘
💡 核心思想

AI不需要我们告诉它每一步怎么做，而是通过看大量例子，自己总结规律。

什么是机器学习？
AI的核心分支

机器学习（ML）是AI的一个核心分支，让计算机通过数据自动学习规律，而不需要我们手写规则。

💡 简单理解

机器学习就像小孩学说话：听了很多话后，自己学会了怎么说，而不是先学语法规则。

🕸️ 神经网络入门

🧠
神经网络是什么？
模仿大脑的结构

神经网络是一种受人类大脑启发的机器学习模型，由很多"神经元"组成。

🧠 神经元结构 vs 神经网络
生物神经元： 人工神经元： ┌─────────────────┐ ┌─────────────────┐ │ 树突输入 │ │ x₁, x₂, ... │ ← 输入 │ ↓ │ │ ↓ │ │ 细胞体处理 │ → │ w₁, w₂, ... │ ← 权重 │ ↓ │ │ ↓ │ │ 轴突输出 │ │ Σ(w·x) │ ← 求和 └─────────────────┘ │ ↓ │ │ 激活函数 │ │ ↓ │ │ y │ ← 输出 └─────────────────┘
💡 简单理解

神经网络就像一个多层的筛选器：数据从一层传到下一层，每层都在提取不同级别的特征。

🔧 常见机器学习算法

算法名称 类型 简单理解 适用场景
线性回归 回归 找一条直线拟合数据 房价预测、销量预测
逻辑回归 分类 用曲线分类两组数据 二分类问题
决策树 通用 像树一样不断分支判断 规则提取、分类
随机森林 集成 多棵树一起投票决定 复杂预测、高精度
K近邻 分类 看邻居来判断自己 推荐系统、简单分类
K均值 聚类 把相似的东西放一起 用户分群、图像压缩

📋 核心概念速查表

概念 解释 简单类比
特征 (Feature) 描述数据的属性 人的身高、体重、年龄
标签 (Label) 我们想要预测的结果 房价高低、是否患病
训练 (Training) 让模型从数据中学习 学生做练习题
模型 (Model) 学习到的规律 解题方法/公式
预测 (Prediction) 用模型输出结果 考试答题
准确率 (Accuracy) 预测对了多少 考试得分率
过拟合 只记住答案，不会应用 背答案但不懂原理
欠拟合 还没学会 题目太难不会做
