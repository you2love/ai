📈 回归 vs 分类

对比项 回归问题 分类问题
输出类型 连续数值 离散类别
例子 预测房价（150万、200万...） 识别猫狗（猫、狗）
输出示例 99.5、150.2 0、1、2 或 猫、狗、鸟
评估指标 MSE、RMSE、MAE 准确率、精确率、召回率

🔧 常见算法

📉
线性回归
Regression

找一条直线拟合数据

输入：面积 → 输出：价格 50㎡ → 100万 80㎡ → 160万 100㎡ → 200万
🚦
逻辑回归
Classification

二分类问题的经典算法

输入：邮件内容 → 输出：是否垃圾 "中奖通知" → 垃圾邮件 ✓ "会议安排" → 正常邮件 ✓
🌳
决策树
Tree

像树一样不断分支判断

天气如何？ / | \ 晴天 阴天 下雨 ✓ ? 取消
🌲
随机森林
Ensemble

多棵树一起投票，精度更高

树1预测：猫 树2预测：狗 树3预测：猫 → 最终结果：猫（2:1）
👥
K近邻
KNN

看邻居是什么，你就是什么

新样本周围3个邻居： 2个是A类，1个是B类 → 分类为A类
📊
K均值
K-Means

自动把相似数据分在一起

用户数据 → 自动分组 年轻+高消费 → 高价值用户

📏 模型评估指标

指标 说明 适用场景
准确率 预测正确的比例 分类问题通用
精确率 预测为正类中真正正类的比例 关注误报
召回率 正类中被正确预测的比例 关注漏报
F1分数 精确率和召回率的调和平均 综合评价
MSE 预测值与真实值差异的平方平均 回归问题
R² 模型解释数据的能力（0-1） 回归问题
💡 学习建议
  1. 先理解每种算法的"直觉"，不要死记公式
  2. 使用Scikit-learn库快速实践各种算法
  3. 完成Kaggle入门竞赛，如Titanic生存预测
  4. 学会使用交叉验证评估模型
