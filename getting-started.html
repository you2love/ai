什么是人工智能？
让机器学会"思考"

人工智能（AI）是让机器具备学习和思考能力的技术。

传统程序 vs AI程序
┌─────────────────────────────────────────────────────────┐ │ │ │ 传统程序： 输入 → 规则 → 输出 │ │ (程序员写好每一步) │ │ │ │ AI程序： 输入 → 学习 → 模型 → 输出 │ │ (机器自己从数据中找规律) │ │ │ └─────────────────────────────────────────────────────────┘

💡 核心思想：不用告诉机器每一步怎么做，给它大量例子，让它自己总结规律。

📱 AI在我们身边

🔊 语音助手

Siri、小爱同学 - 听懂你的话，帮你做事

📷 人脸识别

抖音、小红书 - 猜你喜欢什么

🚗 自动驾驶

特斯拉、萝卜快跑 - 自动识别路况

📝 智能写作

ChatGPT、文心一言 - 帮你写文章

🔍 搜索引擎

百度、Google - 理解你想找什么

📋 入门核心概念

概念 解释 简单类比
数据 (Data) AI学习的原材料 课本里的例题
模型 (Model) 学习到的规律 解题公式
训练 (Training) 从数据中学习 做练习题
预测 (Prediction) 用模型回答新问题 考试答题
准确率 (Accuracy) 预测对了多少 考试得分率
